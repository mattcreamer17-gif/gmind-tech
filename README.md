# GMind.Tech landing page

Bespoke partnership-frame landing page for GMind.Tech, the Boca Raton fintech software studio exhibiting at Booth 1248, eMerge Americas 2026.

Intended URL path: /for/gmind-tech

## What this page does

- Opens with a hook that cites Olga Zhukov's December 2025 Benzinga Fintech Day remark, so the prospect knows homework was done.
- Walks the 60/30/10 frame as a teachable visual.
- Drops the methodology papers (ICM, Ethics Engine) with real anchor links for pressure-testing.
- Closes on 30 minutes with Matt Creamer via Calendly.

## Brand handling

- Tokens lifted directly from the live gmind.tech critical CSS: primary green `#66a538`, accent green `#13ce66`, charcoal `#5b5b5b`, dark bg `#2a2a2a`, foreground `#171717`.
- Typography: Roboto 400 / 500 / 700, loaded from Google Fonts matching GMind's `next/font` setup.
- Logo: `logo.webp` with PNG fallback via `onerror`.
- Eduba appears only in the footer ("Prepared by Eduba, eduba.io") and inside body copy where the partnership context requires it. No Eduba colors or logo in the chrome.

## Files

- `index.html` - single page, no JS.
- `styles.css` - brand tokens, responsive down to 375px, sticky header, dark CTA block.
- `logo.webp`, `logo.png` - copied from `../Brand Pack/assets/`.

## Voice

Written in GMind's register: direct, declarative, period-separated fragments, medium-length sentences, technical but readable. No em dashes, no antithesis, no performed questions, no hype words.
